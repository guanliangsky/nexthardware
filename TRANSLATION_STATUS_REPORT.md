## 🔧 **Automatic Translation Services for Next.js**

### **Option 1: Google Translate Widget** (Free)
**Pros:**
- ✅ Free
- ✅ Easy to implement
- ✅ Supports 100+ languages
- ✅ Automatic translation

**Cons:**
- ❌ Not perfect quality
- ❌ May break layout
- ❌ Not SEO-friendly
- ❌ Generic translations

**Implementation:**
```tsx
// Add to layout.tsx
<Script
  src="//translate.google.com/translate_a/element.js?cb=googleTranslateElementInit"
/>
```

---

### **Option 2: Weglot** (Paid - $10-99/month)
**Pros:**
- ✅ Professional quality
- ✅ SEO-friendly
- ✅ Automatic + manual translation
- ✅ Supports 100+ languages
- ✅ Next.js integration

**Cons:**
- ❌ Paid service
- ❌ Requires API key

**Website:** https://weglot.com

---

### **Option 3: Localazy** (Free tier available)
**Pros:**
- ✅ Free tier (1000 strings)
- ✅ Automatic translation
- ✅ Manual review option
- ✅ Next.js plugin

**Cons:**
- ❌ Limited free tier
- ❌ Requires setup

**Website:** https://localazy.com

---

### **Option 4: i18next + Google Translate API**
**Pros:**
- ✅ Full control
- ✅ Can use Google Translate API
- ✅ Professional setup

**Cons:**
- ❌ Requires API key
- ❌ More complex setup
- ❌ API costs

---

### **Option 5: DeepL API** (Best Quality)
**Pros:**
- ✅ Best translation quality
- ✅ Natural translations
- ✅ API available

**Cons:**
- ❌ Paid (but affordable)
- ❌ Requires API key
- ❌ More setup needed

**Website:** https://www.deepl.com
